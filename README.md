## Retail Food Price Automated Data Analysis & Prediction (Streamlit Edition)

A high-fidelity data intelligence application built with Streamlit. Upload datasets and get instant insights, visualizations, and machine learning predictions with zero code.

## Features
- **Multi-Step Wizard**: Streamlined flow (Upload → Overview → Visuals → Predict).
- **Glassmorphic UI**: Premium dark-mode aesthetics with custom CSS.
- **Smart Parsing**: Supports CSV, Excel, and JSON.
- **Auto-ML**: Automatic task detection (Classification/Regression).
- **Feature Importance**: Visual explanation of prediction drivers.

## Tech Stack
- **Dashboard**: Streamlit (Python)
- **Data & ML**: Pandas, Scikit-learn, Matplotlib, Seaborn

## Run Instructions

### Prerequisites
- Python 3.8+

### Setup
1. Open a terminal in the project directory .
2. (Optional) Create a virtual environment:
   ```powershell
   python -m venv venv
   .\venv\Scripts\activate
   ```
3. Install dependencies:
   ```powershell
   pip install -r requirements.txt
   ```

### Execution
Run the application using Streamlit:
```powershell
streamlit run app.py
```
The application will open automatically in your browser (usually at `http://localhost:8501`).
